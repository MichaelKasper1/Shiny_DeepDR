Find Sample

Module 2 (Find Sample): Finding a cancer sample with similar genomic features to those of a query sample

Module 2 searches for cell lines in the CCLE or tumors of TCGA that have similar genomic features to the query sample and display their drug responses. It cross-references to real and DeepDR-predicted drug response data of top similar CCLE samples, or predicted data of top similar TCGA samples. To start the analysis, the user should upload the genomic data of the query sample (mutations alone, expression alone, or mutations plus expression), and select the sample type to analyze (cell line: CCLE; tumor: TCGA). After specifying the file location(s), the user should click on the purple button [SUBMIT] to start the analysis. Alternatively, the user can test our tool using a built-in example sample by simply clicking on the blue icon [RUN EXAMPLE].

Input Data Format

Please visit the "Input Data Format" help page for detailed information regarding data preparation and downloading example files.

Similarity Analysis based on genomic profiles

  • Mutations: Given a query sample and a specified sample type, our tool searches through all 704 cell lines of the CCLE or 9,059 tumors of TCGA with paired mutation and expression data. The similarity between the query sample and samples in our database is measured and ranked by the Jaccard index for mutated genes among 18,281 genes.
  • Gene expression:  The similarity between the query sample and samples in our database is measured and ranked by the Pearson correlation coefficients among 15,363 genes for gene expression data.
  • Paired mutation and expression:  When sample-paired data are uploaded, the tool averages the ranks of mutations and gene expression to determine the overall ranking of ranks CCLE or TCGA samples. 

Outputs

  • Panel 2-1: Similar cell lines/tumors to the query sample identified based on Jaccard index (mutations) and/or Pearson correlation (gene expression) and corresponding clinical information. 
  • Panel 2-2: Network of similar samples and their top effective drugs.
