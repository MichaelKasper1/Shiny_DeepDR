Find Drug

Module 1 (Find Drug): Finding drugs that are predicted to inhibit a query sample

Module 1 uses DeepDR to predict the query sample's response to 265 FDA-approved or investigational anti-cancer compounds. To start the analysis, the user should upload the genomic data of the query sample (mutations alone, expression alone, or mutations plus expression), and select the sample type to analyze (cell line: CCLE; tumor: TCGA). After specifying the file location(s), the user should click on the purple button [SUBMIT] to start the analysis. Alternatively, the user can test our tool using a built-in example sample by simply clicking on the blue icon [RUN EXAMPLE].

Input Data Format

Please visit the "Input Data Format" help page for detailed information regarding data preparation and downloading example files.

Running DeepDR Prediction

We implemented three versions of DeepDR that take different sets of omics data:
  • Mutation + expression (recommended): Original DeepDR model that integrates mutations and gene expression to predict the cancer sample's response (measured by IC50 values) to 265 compounds. This model was used to derive all major findings from the DeepDR publication (Chiu et al. BMC Medical Genomics 2019). The model achieved better performance (per-sample correlation) than alternatively models built on single omics. Per-drug performance in testing samples are provided with our tool output.
  • Mutation alone: Alternative DeepDR model that takes only gene mutation data. The model was constructed by removing the expression component (i.e., encoder) from DeepDR, with all other training parameters being the same. Per-drug performance in testing samples are provided with our tool output.
  • Expression alone: Alternative DeepDR model that takes only gene expression data. The model was constructed by removing the mutation component (i.e., encoder) from DeepDR, with all other training parameters being the same. Per-drug performance in testing samples are provided with our tool output.
